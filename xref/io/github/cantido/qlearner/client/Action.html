23   * <p>
24   * The {@link Agent} assumes that taking gridworld.actions using {@link Action#run()} somehow
25   * changes the state of the {@link Environment}. Calling {@code execute()} on {@code Action}s is the
26   * "output" of this system. This could drive motors, change an HVAC's set temperature, move a game

28   * {@link Environment#getState()}, then the system can learn to optimize its choices.

35   * everywhere!

39   * new threads, be careful of the QLearning system interacting with the {@link Environment} is

41   * next State from it.
